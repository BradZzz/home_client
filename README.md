## Technology

React, Nodejs, MongoDB

## Installation

Create a .env with the following variables defined:

NODE_ENV
DOM_USR
DOM_PASS
MONGODB
DOM_INDEX
ESTIBOT_USR
ESTIBOT_PASS

Then start the server like so:

```
  npm start
```

This should kick off the installation process right away, but if there is a problem, you can just use:

```
  npm install
```

## Building

```
  npm run build
```
